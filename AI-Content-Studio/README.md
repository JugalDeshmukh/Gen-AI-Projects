# 🚀 AI Content Studio

An AI-powered content generation application built using **Python**, **Streamlit**, and **Groq API (Llama 3.3)**. The application enables users to generate high-quality content such as LinkedIn posts, professional emails, blog outlines, resume summaries, and cover letters through an intuitive web interface.

---

## 📌 Features

- Generate AI-powered content in seconds
- Multiple content types:
  - LinkedIn Posts
  - Professional Emails
  - Blog Outlines
  - Resume Summaries
  - Cover Letters
- Select different writing tones
- Choose output length (Short, Medium, Long)
- User-friendly Streamlit interface
- Powered by Groq API with Llama 3.3 for fast inference

---

## 🛠 Tech Stack

- Python
- Streamlit
- Groq API
- Llama 3.3
- Prompt Engineering

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/JugalDeshmukh/Gen-AI-Projects.git
```

Navigate to the project folder

```bash
cd Gen-AI-Projects/AI-Content-Studio
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
